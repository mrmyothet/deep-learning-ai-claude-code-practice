# Claude Code: A Highly Agentic Coding Assistant

## Claude Code Installation

To get started with the lessons, follow these steps:

- **Prerequisite:** Ensure you have **Node.js** installed on your system.
- **Installation Command:** Run `npm install -g @anthropic-ai/claude-code` in your terminal.
- **Guides:** Refer to the course's original documentation for specific [Installation Guides](https://docs.anthropic.com/en/docs/agents-and-tools/claude-code/overview) and the [Windows Setup](https://docs.anthropic.com/en/docs/agents-and-tools/claude-code/overview) if applicable.
- **Launching:** Type `claude` in your terminal or within the VS Code integrated terminal.
- **Troubleshooting:** If the `claude` command is not recognized in VS Code, ensure the `code` command is installed in your PATH. Use `Cmd+Shift+P` (Mac) or `Ctrl+Shift+P` (Windows/Linux) and search for "Shell Command: Install ‘code’ command in PATH".
